## ✅ Key Features

- Loan contract management: Create / Edit / Print PDF  
- **Automatic period scheduling based on loan date and terms** - **Color-coded alerts: Red (Overdue), Yellow (Upcoming), Green (Due today)** - Automatic interest calculation per period or days from the last period  
- Records payment history and supports smart loan extensions  
- Diacritic-insensitive search: filter by name, code, asset type, status, and date range  
- Status management: Active, Upcoming, Overdue, Settled, Redeemed  
- Print contracts and payment history via PDF (QuestPDF)  
- **Statistics: Total interest collected, total active contracts, total outstanding balance** *(v1.6: current month focus)* - Permanent hardware-locked (HWID) licensing system with encryption  
- Supports international currency and date formats (`dd-MM-yyyy`)  
- Modern rounded UI, optimized for Full HD (1920x1080)
